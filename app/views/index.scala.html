Pathfinder

Real-time logistics routing for mobile applications.

Let us do the math. You focus on your users.

Pathfinder is a logistics platform for shared transportation and delivery applications. We provide real-time route computation and updates as transportation requests come in and vehicles go on and offline.

Native client libraries for popular platforms.

Instead of programming directly against our websocket API, you can use our developer-friendly client libraries to interact with the Pathfinder platform.

Supported Platforms
Android Docs Maven
iOS Docs Cocoapods
Javascript Docs Bower

Real-time updates

Transportation requests, vehicle creation, status changes and route updates are pushed to clients via websockets in real-time.

Cluster separation

Transports and commodities are grouped into clusters that are routed separately. This gives you granular control over the routes for your application.

Route configuration

The admin dashboard allows you to configure capacities, parameters and custom objective functions that configure the route computation process.

Get started now. Yes, that means you.

To get your app up and running fast, we've created several tutorials on using Pathfinder as well as a from-scratch sample app. You can check these out in our comprehensive documentation suite.
